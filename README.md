# test-skill

本仓库用于存放个人平时编写和积累的各类 **Skill**（技能/能力模块）。

## 简介

这里的每一个 Skill 都是可复用、可组合的独立能力单元，涵盖自动化测试、工程搭建、工具脚本等多个方向。目的是将日常工作中沉淀的经验和流程标准化，方便在不同项目间快速复用。

## 现有 Skill 列表

| Skill 名称 | 说明 |
|-----------|------|
| `auto-test-project-builder` | 自动化测试工程搭建脚手架，支持根据前端/后端项目一键生成测试工程 |

## 如何使用

每个 Skill 通常是一个独立文件夹，内含：
- `SKILL.md`：该 Skill 的详细说明、使用方式和示例
- 相关脚本或模板文件

进入对应 Skill 目录，阅读其 `SKILL.md` 即可了解具体用法。

## 持续更新

本仓库会随日常工作和学习持续补充新的 Skill，欢迎参考和交流。
